## [Unreleased]

### Added
- Fixed parallax forest background on homepage with scroll-over effect
- Alternating section backgrounds: solid cream and translucent with forest visible
- PageCta component with subtle forest-green tint
- Full logo lockup on all screen sizes (navbar)
- Scroll-to-top button visible on all screen sizes
- Credential badges: 3-column centered grid layout on mobile

### Changed
- Hero gradient overlays recalibrated for better text readability on tablet and mobile
- Section background system: odd sections = solid cream, even sections = translucent
- Navbar tablet layout tightened to prevent CTA button overflow
- Footer and FAQ section use explicit solid backgrounds above parallax layer

### Fixed
- Navbar active link underline flicker (removed Framer Motion shared layout animation)
- Hero gradient obscuring right half content on tablet
- Scroll-to-top button overlapping WhatsApp floating button on mobile
- Fixed forest background showing through solid sections (z-index stacking fix)
